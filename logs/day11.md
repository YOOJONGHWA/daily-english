# 📘 Day 11: Clean Code 영어 원서 독해

## 문장

> "Avoid names that are misleading or too clever."

---

## 해석

> **오해를 불러일으키거나 지나치게 똑똑한(재치 있는) 이름은 피하라.**

---

## 구조 분석

- **명령문 동사**: Avoid (피하라)
- **목적어**: names
- **관계절**: that are misleading or too clever

---

## 단어 정리

| 표현       | 뜻                |
| ---------- | ----------------- |
| avoid      | 피하다            |
| misleading | 오해하게 만드는   |
| clever     | 영리한, 재치 있는 |

---

## 독해 규칙 체크

- [ ] ① 주어 + 동사 먼저 찾기 → (명령문, 주어 생략된 문장)
- [x] ② 앞에서부터 순서대로 읽기
- [x] ③ 수식어 생략하고 핵심 보기
- [ ] ④ 조건어 없음
- [x] ⑤ 동사 중심으로 보기

---

## 내일 문장 미리보기

> "Prefer names that reveal intent."
