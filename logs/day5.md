# 📘 Day 5: Clean Code 영어 원서 독해

## 문장

> "We make the mess, and then we clean it up under pressure."

---

## 해석

> 우리는 문제를 만들고, 그리고 나서 압박 속에서 그것을 해결한다.

---

## 구조 분석

- **주어 + 동사**: We make / We clean
- **부사구**: under pressure

---

## 단어 정리

| 표현           | 뜻               |
| -------------- | ---------------- |
| make a mess    | 문제를 만들다    |
| clean up       | 치우다, 정리하다 |
| under pressure | 압박을 받으며    |

---

## 독해 규칙 체크

- [x] ① 주어 + 동사 먼저 찾기
- [x] ② 앞에서부터 순서대로 읽기
- [x] ③ 수식어 생략하고 핵심 보기
- [ ] ④ 조건어 없음
- [x] ⑤ 동사 중심으로 보기

---

## 내일 문장 미리보기

> “We want to be clear about our work.”
