# 📘 Day 3: Clean Code 영어 원서 독해

## 문장

> "There are two reasons why code might not be clean."

---

## 해석

> 코드가 깔끔하지 않은 데에는 두 가지 이유가 있다.

---

## 구조 분석

- **문장형태**: there are + 명사
- **핵심**: two reasons
- **수식**: why code might not be clean

---

## 단어 정리

| 단어           | 뜻              |
| -------------- | --------------- |
| there are      | ~이 있다        |
| reasons        | 이유들          |
| why            | ~하는 이유      |
| might          | ~일지도 모른다  |
| (not) be clean | (깔끔하지) 않다 |

---

## 독해 규칙 체크

- [x] ① 주어 + 동사 먼저 찾기
- [x] ② 앞에서부터 순서대로 읽기
- [x] ③ 수식어 생략하고 핵심 보기
- [ ] ④ 조건어 없음
- [x] ⑤ 동사 중심으로 보기
