# 📘 Day 23: Clean Code 영어 원서 독해

## 📖 문장

**"Use pronounceable names."**

---

## 📌 해석

발음할 수 있는 이름을 사용하라.  
(즉, 사람들이 읽고 말하기 쉬운 이름을 지어야 한다.)

---

## 🔎 구조 분석

- **동사(명령문):** Use
- **목적어:** pronounceable names

---

## 🗂 단어 정리

| 표현              | 뜻             |
| ----------------- | -------------- |
| **use**           | 사용하다       |
| **pronounceable** | 발음할 수 있는 |
| **names**         | 이름들         |

---

## ✅ 독해 규칙 체크

1. 주어 + 동사 (명령문이므로 주어 **you** 생략)
2. 동사(**use**) → 목적어(**pronounceable names**)
3. 앞에서부터 순서대로 해석
4. 조건어, 수식어 없음
5. 명확하고 단순한 문장 구조
