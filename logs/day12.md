# 📘 Day 12: Clean Code 영어 원서 독해

## 문장

> "Prefer names that reveal intent."

---

## 해석

> 의도를 드러내는 이름을 **선호하라.**

---

## 구조 분석

- **동사 (명령문)**: Prefer (선호하라)
- **목적어**: names
- **관계절**: that reveal intent

---

## 단어 정리

| 표현   | 뜻                 |
| ------ | ------------------ |
| prefer | 선호하다           |
| reveal | 드러내다, 보여주다 |
| intent | 의도               |

---

## 독해 규칙 체크

- [ ] ① 주어 + 동사 먼저 찾기 → (명령문, 주어 생략)
- [x] ② 앞에서부터 순서대로 읽기
- [x] ③ 수식어 생략하고 핵심 보기
- [ ] ④ 조건어 없음
- [x] ⑤ 동사 중심으로 보기
