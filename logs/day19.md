# 📘 Day 19: Clean Code 영어 원서 독해

## 문장

> "Bad names introduce confusion and slow down development."

---

## 해석

> 나쁜 이름은 **혼란을 불러일으키고 개발을 늦춘다.**

---

## 구조 분석

- **주어**: Bad names
- **동사**: introduce / slow down
- **목적어**: confusion / development

---

## 단어 정리

| 표현        | 뜻                     |
| ----------- | ---------------------- |
| bad names   | 나쁜 이름              |
| introduce   | 도입하다, 불러일으키다 |
| confusion   | 혼란                   |
| slow down   | 늦추다, 속도를 줄이다  |
| development | 개발                   |

---

## 독해 규칙 체크

- [x] ① 주어 + 동사 먼저 찾기
- [x] ② 앞에서부터 순서대로 읽기
- [x] ③ 수식어 생략하고 핵심 보기
- [ ] ④ 조건어 없음
- [x] ⑤ 동사 중심으로 보기

---

## 내일 문장 미리보기
