# 📘 Day 25: Clean Code 영어 원서 독해

## 📖 문장

**"Avoid disinformation."**

---

## 📌 해석

잘못된 정보를 피하라.  
(즉, 변수명이나 함수명이 혼동되거나 잘못된 의미를 주지 않도록 해야 한다.)

---

## 🔎 구조 분석

- **동사(명령문):** Avoid
- **목적어:** disinformation

---

## 🗂 단어 정리

| 표현               | 뜻                     |
| ------------------ | ---------------------- |
| **avoid**          | 피하다, 방지하다       |
| **disinformation** | 잘못된 정보, 허위 정보 |

---

## ✅ 독해 규칙 체크

1. 주어 + 동사 (명령문이므로 주어 **you** 생략)
2. 동사(**avoid**) → 목적어(**disinformation**)
3. 앞에서부터 순서대로 해석
4. 조건어, 수식어 없음
5. 짧고 직설적인 명령문
